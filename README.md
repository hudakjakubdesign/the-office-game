# The Office Game

A playable Grayling-branded top-down pixel-art office demo. Walk through a compact corporate office and meet colleagues from Finance, Design & Marketing, and Brand Portfolio.

## Run locally

1. Install Node.js 22 or newer.
2. Run `npm install`.
3. Run `npm run dev`.
4. Open `http://localhost:3000`.

## Controls

- Move with WASD or the arrow keys.
- Talk with E, Enter, or Space.
- Touch controls are included for mobile devices.

The first version uses original placeholder pixel artwork so the office, characters, dialogue, and branding can be refined in later rounds.
